BEL SEKOLAH OTOMATIS
====================
variabel:

HARI 		-> berisi nama hari (MINGGU, SENIN, SELASA, RABU, KAMIS, JUMAT, SABTU)
TANGGAL 	-> format dd/mm/yyyy (17/06/2020)
WAKTU		-> format hh:mm (07:00)
KODE JADWAL	-> berisi jam pelajaran yang berbeda-beda(OFF, A, B, C, D, E, F)
JAM KE		-> urutan jam pelajaran (1 sampai 20)
NADA		-> nomor track yang diputar pada saat jam pelajaran tertentu (1 sampai 30)
TRACK		-> nomor track yang sedang diputar saat tes suara(1 sampai 30)
VOL			-> besar volume suara (0 sampai 30)


-> mampu menyimpan 6 macam penjadwalan(dengan kode: "A, B, C, D, E, F") yang berbeda (misalnya digunakan secara khusus untuk hari SENIN, JUMAT, SABTU, jadwal saat UTS/UAS/PUASA/CLASSMEETING, dll)
-> dalam 1 hari maksimal menyimpan 20 perubahan jam pelajaran meliputi: jam masuk/upacara/istirahat/pergantian jam pelajaran/pulang. (suara bel dapat disesuaikan pada masing-masing pergantian jam)
-> file suara yang bisa diatur sesuai kebutuhan dan disimpan pada microSD dengan format mp3
-> 


PENGATURAN AWAL:
-> atur file suara pada microSD, maksimal 30 macam file suara yang berbeda; simpan dengan nama file 0001-0030 (contoh: 0002.mp3 / 0010.mp3 / 0024.mp3)
-> atur jadwal:  1> masukkan jam pelajaran pada tiap KODE JADWAL sesuai dengan kebutuhan(jika dibutuhkan 3 macam jadwal yang berbeda dalam sepekan, maka aturlah kode jadwal A, B, dan C); 
					aturlah "JAM dan MENIT" ketika bel harus berbunyi kemudian atur SUARA yang dikeluarkan pada saat "JAM dan MENIT" tersebut.
				 2> sesuaikan masing-masing hari dengan kode jadwal yang telah di buat (misal hari SENIN dengan kode A; SELASA-KAMIS dengan kode B; JUMAT dengan kode C)


TAMPILAN AWAL:
tampilkan, hari, tanggal & waktu (jam, menit, detik)
tampilkan kode jadwal & status jam pelajaran


TEKAN dan TAHAN Tombol:
-> 1 <- masuk mode BEL MANUAL 
		bel dibunyikan secara manual dengan menekan tombol angka 0-9(setiap tombol yang ditekan akan mengeluarkan suara yang berbeda); 
		
-> 2 <- masuk menu PENGATURAN
		terdapat 4 pilihan:
		1> pengaturan "waktu"  -> sesuaikan TANGGAL dan JAM saat ini 
		
		2> pengaturan "hari"   -> sesuaikan HARI saat ini
		
		3> pengaturan "kode"   -> menyesuaikan masing-masing HARI dengan KODE JADWAL
		
		4> pengaturan "jadwal" -> 
		
-> 3 <- masuk mode TES SUARA
